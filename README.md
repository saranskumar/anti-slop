# Antigravity Developer OS

This Antigravity workspace is a developer-first and designer-capable operating system for building, reviewing, securing, and refining real software.

## Canonical surfaces
- `rules/`: always-on guardrails for secure, maintainable engineering work
- `knowledge/`: reusable engineering and design heuristics
- `skills/`: focused developer and designer capabilities with precise routing descriptions
- `global_workflows/`: multi-step execution pipelines that orchestrate several skills
- `archive/`: material kept only for reference during cleanup

## System philosophy
- one lightweight intake skill for broad requests
- otherwise developer and designer skills only
- secure defaults
- explicit trust boundaries
- multi-language and multi-runtime support
- production-minded outputs over ideation-heavy artifacts

## Primary supported domains
- frontend applications
- backend services and APIs
- Python applications, CLIs, scripts, and workers
- TypeScript and JavaScript backends
- Supabase and Firebase systems
- databases, auth, integrations, and environment-specific deployments
