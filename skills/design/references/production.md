# Production

## Export and delivery

- choose formats based on destination: web, print, document, or asset pipeline
- preserve source files that keep typography, color, and layout editable

## Print versus digital

- confirm color mode and output size before export
- expect physical output differences from screen previews

## Font licensing

- track licensed families used in production
- confirm embedding and distribution rights where relevant

## File preparation

- keep naming consistent
- separate source files from exports
- avoid shipping flattened assets when editable masters are required
