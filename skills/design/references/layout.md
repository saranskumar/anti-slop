# Layout

## Grid systems

- use a clear grid even when it stays invisible
- let the grid drive alignment, spacing, and grouping
- break the grid only on purpose

## Alignment rules

- align repeated elements consistently
- preserve shared edges when scanning matters
- avoid decorative misalignment that harms comprehension

## Spacing systems

- use consistent spacing increments
- separate unrelated groups with larger gaps
- keep internal spacing tighter than external spacing

## Hierarchy through structure

- primary information should occupy the clearest visual zone
- secondary information should support rather than compete
- empty space should reduce noise, not create randomness
