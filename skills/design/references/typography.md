# Typography

## Font classification

- Serif: strong for editorial or traditional reading contexts
- Sans-serif: preferred for modern UI and digital surfaces
- Slab serif: strong, dense headline emphasis
- Script: accent-only, never long-form reading
- Decorative: special-purpose display use only

## Usage rules

- limit families to one or two
- define heading, subheading, body, caption, and metadata levels
- keep body text readable at the actual medium size
- choose tone to match message, audience, and context

## Readability

- keep line height comfortable for body copy
- avoid overly tight tracking in dense text
- do not rely on weight changes alone for hierarchy
- test readability on the real surface, not just in design view

## Variable fonts

- useful for UI flexibility and reducing multiple file loads
- still require intentional weight choices and fallback planning

## Licensing

- verify commercial use
- avoid personal-use-only fonts in production
- document licensed families used in the system
