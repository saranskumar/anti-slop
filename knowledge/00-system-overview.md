# System Overview

This workspace is a developer/designer operating system for secure software delivery. It is optimized for building, reviewing, refining, hardening, and maintaining real software systems rather than for startup ideation.

## Primary responsibilities
- convert broad requests into implementation requirements
- plan architecture across frontend, backend, auth, data, and environments
- design services, APIs, jobs, scripts, CLIs, and integrations
- support Python, JavaScript, TypeScript, SQL, shell, serverless, and multi-environment work
- produce implementation-relevant UX and frontend outputs

## Skill philosophy
- one intake skill for broad requests
- many precise specialist skills for execution work
- workflows orchestrate several specialist skills when needed
