# Product Context

This product, **00antigravity**, is a professional-grade **AI Operating System**. It is a structured framework of rules, skills, and context documents designed to enable high-fidelity reasoning and production-grade execution for AI agents (specifically Antigravity) across any technical stack.

## Primary product goals
- **Operational Reliability**: Producing consistent, verified, and high-quality outputs.
- **Deterministic Execution**: Moving away from ad-hoc prompting and toward repeatable GSD (Get Shit Done) workflows.
- **Extreme Context Management**: Using skills and project memory to eliminate knowledge decay.
- **Stack Agnostic Excellence**: Designing principles that work across any technology while providing deep, opinionated implementations when requested.
- **Premium Documentation**: Ensuring the system is self-documenting and always synchronized with its active state.

## Core Pillars
- **Rule Engine**: 19 global rules enforcing safety, quality, and standards.
- **Skill Library**: 53+ specialized modules for specific technical and business tasks.
- **Workflow Orchestration**: Chainable global workflows that guide the agent from idea to deployment.
- **Memory Layer**: Persistent session logs and architecture decision records.
